# Bike Sales Interactive Dashboard (Excel)

## Project Overview
This project focuses on analyzing customer demographics to identify patterns in bike purchasing behavior. I transformed a raw dataset of 1,000+ records into a dynamic, interactive dashboard to help stakeholders visualize key sales drivers.

## Data Cleaning Process
- **Duplicate Removal:** Identified and removed redundant entries to ensure data integrity.
- **Data Standardization:** Used Find & Replace to convert short-hand codes (M/S, M/F) into readable categories (Married/Single, Male/Female).
- **Custom Categorization:** Created a nested `IF` function to generate **Age Brackets** (Adolescent, Middle Age, Old) for clearer demographic grouping.
- **Currency & Date Formatting:** Fixed data type inconsistencies to ensure accurate mathematical calculations.

## Final Dashboard Features
- **Key Metrics:** Visualized Average Income per Purchase and Customer Age Brackets.
- **Trend Analysis:** Created a line chart to show the peak purchasing age (31–54).
- **Interactivity:** Integrated **Slicers** (Marital Status, Region, Education) that allow users to filter the entire dashboard with a single click.

## Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Advanced Formulas, XLOOKUP)
